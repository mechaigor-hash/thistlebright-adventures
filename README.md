# Thistlebright Adventures

![Cover](art/generated/01-cover-scottish-fairy.png)

A DND-inspired, high-fantasy tabletop RPG for **ages 5–7** with a cozy Scottish fairy mythological theme.

## Image-heavy draft
This repo is intentionally art-forward:
- Images on every major book/page section.
- Dedicated image for **every creature in the bestiary**.
- Reproducible prompts stored in `art/prompts/`.
- Generated art stored in `art/generated/`.

## Core mechanic
Roll **1d6 + a simple stat**. A total of **5+ succeeds**. On 3–4, success with a silly twist. On 1–2, try a new plan or get help from a friend.

## Safety / age design
No gore, no horror, no permanent death, simple choices, cooperative problem solving, reading-light mechanics, 20–40 minute sessions.
