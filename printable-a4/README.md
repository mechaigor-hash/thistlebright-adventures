# Printable A4 Portrait Versions

Open these HTML files in a browser and print to PDF using **A4 Portrait**.

This version uses a more integrated tabletop-book layout:

- parchment-style page backgrounds and framed artwork
- race/class portraits anchored inside their description cards rather than floating as separate image blocks
- class abilities/spells in inset feature boxes
- bestiary entries wrapped into side-art creature cards
- `break-inside: avoid` / `page-break-inside: avoid` rules on figures, tables, callouts, and cards to prevent artwork from splitting across pages in print preview

Files:

- `player-handbook-a4.html`
- `dungeon-master-guide-a4.html`
- `bestiary-a4.html`
- `example-campaigns-a4.html`
- `combined-book-a4.html`
